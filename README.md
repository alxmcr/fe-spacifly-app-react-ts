## Run Locally

Clone project

```bash
  git clone https://github.com/alxmcr/spacifly-app
```

Go to project directory

```bash
  cd spacifly-app
```

Install NPM dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

Open the browser:

```
http://localhost:3000
```
